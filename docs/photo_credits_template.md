# Szablon uzupełniania zdjęć

Dla każdego gatunku wypełnić:

| Pole | Opis |
|---|---|
| `image` | lokalna ścieżka, np. `./images/okratek-australijski.webp` |
| `image_alt` | opis alternatywny zdjęcia |
| `image_author` | autor / właściciel praw |
| `image_source` | link do strony pliku, nie tylko do samego JPG |
| `image_license` | np. CC BY-SA 4.0, CC BY 4.0, Public Domain |
| `license_url` | link do warunków licencji |
| `image_changes` | np. „skadrowano i skompresowano do WebP” |

Przy zdjęciach z Wikimedia Commons używać strony pliku jako źródła i sprawdzić licencję osobno dla każdego zdjęcia.
